Back-end Projects

The largest project I have worked on to-date is a complete search engine. It contains a fully functioning web crawler and search suggestions. The web crawler is coded in C# and uses aworker role on Azure cloud instances to crawl sites and save the data it collects to Azure Table. A cloud queue is used to facilitate communication between the worker role and the web role which houses the actual search page, as well as the crawler dashboard which can send commands to the crawler. Search suggestions are provided by taking a list of ~10 million words and phrases and storing them in a Trie. This allows for extremely quick results. An AWS RDS server was also used to return data about specific search requests through SQL queries in PHP PDO.
